# Landing Page — Starter

Prosty, responsywny landing page starter. Zawiera:

- index.html
- styles.css
- script.js

Instrukcje:

1. Otwórz index.html w przeglądarce lub opublikuj przez GitHub Pages.
2. Dostosuj treść i styl w plikach powyżej.

Deployment (GitHub Pages):
- W ustawieniach repo, włącz GitHub Pages dla branch `main` i folder `/`.
